## Phiên bản Tiếng Việt (Vietnamese Version)

UniCou là một dự án website tĩnh (front-end) dành cho nền tảng học trực tuyến. Dự án cung cấp giao diện người dùng hiện đại, hiển thị tốt trên mọi thiết bị (responsive), tích hợp các tính năng như chuyển đổi giao diện (tối/sáng), đổi màu chủ đề và hiệu ứng kính (glassmorphism).

### 🚀 Tính năng nổi bật
- **Trang chủ (Home):** Hiển thị banner, thông tin nổi bật, danh sách khóa học và đánh giá của học viên.
- **Khóa học (Courses):** Liệt kê các khóa học chuyên ngành (AD, DA, NS, BA) cùng thông tin chi tiết của mỗi khóa học.
- **Tài khoản:** Cung cấp giao diện Đăng nhập (Log-in) và Đăng ký (Sign-up).
- **Liên hệ (Contact):** Giao diện biểu mẫu liên hệ hỗ trợ (bao gồm file xử lý `contact.php`).
- **Tùy chỉnh giao diện (Style Switcher):** Người dùng có thể tự đổi màu sắc chủ đề, bật/tắt chế độ tối (Dark Mode) và hiệu ứng thủy tinh (Glass Effect).

### 🛠 Thư viện và Công nghệ sử dụng
- **HTML5 & CSS3:** Xây dựng cấu trúc và định dạng cho giao diện.
- **JavaScript (Vanilla):** Xử lý các hiệu ứng động, tương tác và tính năng cài đặt giao diện (Style Switcher).
- **Bootstrap 5.3.0:** Framework CSS để xây dựng layout lưới responsive nhanh chóng.
- **FontAwesome 6.4.0:** Bộ icon sử dụng trên toàn bộ trang web.

### 📋 Yêu cầu hệ thống (Requirements)
Vì dự án chủ yếu là Front-end tĩnh, nên không yêu cầu cài đặt môi trường phức tạp:
- Trình duyệt web bất kỳ bản cập nhật mới nhất (Google Chrome, Firefox, Safari, Edge, v.v.).
- *(Tùy chọn)* Máy chủ web ảo (như XAMPP, hoặc Live Server trên VSCode) nếu bạn muốn chạy file `contact.php` hoặc kiểm tra code mượt mà hơn.

### ⚙️ Hướng dẫn cách chạy (How to run)
1. Tải toàn bộ mã nguồn về máy hoặc dùng lệnh `git clone`.
2. Mở thư mục dự án vừa tải về.
3. Click đúp chuột vào file `index.html` để mở trang web trực tiếp trên trình duyệt.
4. **Khuyên dùng:** Nếu bạn có sử dụng Visual Studio Code, hãy cài đặt extension **Live Server**. Sau đó nhấn chuột phải vào file `index.html` -> chọn "Open with Live Server" để trải nghiệm đầy đủ tính năng nhất.

## English Version

UniCou is a static website (front-end) project for an online learning platform. The project provides a modern, fully responsive user interface, integrating features such as a style switcher (light/dark mode), theme color change, and glassmorphism effects.

### 🚀 Key Features
- **Home:** Displays banners, highlights, course lists, and student testimonials.
- **Courses:** Lists specialized courses (AD, DA, NS, BA) with detailed information for each course.
- **Account:** Provides user interfaces for Log-in and Sign-up.
- **Contact:** A support contact form interface (includes `contact.php` processing file).
- **Style Switcher:** Users can customize theme colors, toggle Dark Mode, and enable Glass Effects.

### 🛠 Libraries and Technologies
- **HTML5 & CSS3:** For building structure and interface styling.
- **JavaScript (Vanilla):** For handling animations, interactions, and the Style Switcher features.
- **Bootstrap 5.3.0:** A CSS framework for quickly building responsive grid layouts.
- **FontAwesome 6.4.0:** An icon set used throughout the website.

### 📋 System Requirements
Since this project is primarily a static front-end application, it does not require a complex environment setup:
- Any modern, up-to-date web browser (Google Chrome, Firefox, Safari, Edge, etc.).
- *(Optional)* A local web server (like XAMPP or Live Server in VSCode) if you want to run the `contact.php` file or test the code more smoothly.

### ⚙️ How to run
1. Download the entire source code to your machine or use the `git clone` command.
2. Open the downloaded project folder.
3. Double-click the `index.html` file to open the website directly in your browser.
4. **Recommended:** If you are using Visual Studio Code, install the **Live Server** extension. Then right-click on the `index.html` file -> select "Open with Live Server" for the best experience.
